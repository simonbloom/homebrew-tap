# Homebrew Tap for Progress11

This is the official Homebrew tap for [Progress11](https://github.com/simonbloom/gittool-releases), a native macOS Git client.

## Installation

```bash
brew tap simonbloom/tap
brew install --cask progress11
```

Or in a single command:

```bash
brew install --cask simonbloom/tap/progress11
```

## Updating

```bash
brew upgrade --cask progress11
```

## Uninstalling

```bash
brew uninstall --cask progress11
```
